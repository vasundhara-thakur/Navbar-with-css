#  Navigation Bar UI

This project is a clean **Navigation Bar** built using HTML & CSS.  
It focuses on modern UI interactions and introduces new CSS concepts to create a smooth and dynamic design.

---

##  Features

- Simple navigation bar  
- Smooth hover animation on menu items  
- Interactive highlight effect using **CSS Anchor Positioning**  
- Uses **Flexbox** for clean alignment and layout spacing  
- Styled with gradients, transitions, and blend effects  

---

##  What I Learned

- How to use **CSS Anchor Positioning** for linked UI effects  
- Applying **mix-blend-mode** for creative hover designs  
- Creating smooth animations using CSS transitions  
- Managing layout structure with Flexbox 

---

##  Tech Stack

- **HTML5**  
- **CSS3**
